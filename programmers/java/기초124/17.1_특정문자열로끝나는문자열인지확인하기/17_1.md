# ⭐ lastIndexOf()

- 특정 문자열(또는 문자)이 마지막으로 등장하는 위치(인덱스)

```java
문자열.lastIndexOf(찾을값)
```
```java
String s = "AbCdEFG";
System.out.println(s.lastIndexOf("dE")); // 3

// A b C d E F G
// 0 1 2 3 4 5 6
//       ↑
```

# ⭐ substring()

- 문자열의 일부를 잘라서 새로운 문자열 생성

```java
문자열.substring(시작인덱스, 끝인덱스)
```
```java
String s = "AbCdEFG";
System.out.println(s.substring(0, 5)); // AbCdE

// A b C d E F G
// 0 1 2 3 4 5 6
// ↑         ↑
// 0         5 (미포함)
```
