# ⭐ contains()

- 문자열에 특정 문자열이 포함되어 있는지

```java
"programmers".contains("gram"); // true
"programmers".contains("abc");  // false
```

# ⭐ add()

- java 에서 list 에 요소를 추가할떄 사용
- (python 에서는 append 를 사용)
- java 에서는 string 을 만들때 append 사용

```java
List<String> list = new ArrayList<>();
list.add("apple");
list.add("banana");
// ["apple", "banana"]
```

# ⭐ toArray()

- List → 배열 변환

```java
String[] arr = list.toArray(new String[0]);
```
