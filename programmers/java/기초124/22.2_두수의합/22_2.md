# ⭐ BigInteger

- 클래스 기반 무한 정밀도 정수
- 메모리가 허용하는 한 자릿수 제한 없음

| 타입         | 최대 크기 |
| ---------- | ----- |
| int        | 약 21억 |
| long       | 약 9경  |
| BigInteger | 제한 없음 |

```java
import java.math.BigInteger;
```

- 정수로 생성
- 문자열로 생성 (가장 많이 씀)
    - 가장 안전
    - 코테에서 표준

```java
BigInteger b = BigInteger.valueOf(12345);
BigInteger a = new BigInteger("12345678901234567890");
```

### 코테에서 BigInteger 쓰는 대표 상황

- 숫자 길이가 100자리 이상
- 문자열로 큰 수 덧셈 / 곱셈
- 오버플로우가 절대 나면 안 되는 문제
