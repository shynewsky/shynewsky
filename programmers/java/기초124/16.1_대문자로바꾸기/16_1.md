# ⭐ toUpperCase()

- 문자열에 포함된 모든 알파벳을 대문자로 변환해서
새로운 문자열(String) 을 반환하는 메서드

```java
String result = myString.toUpperCase();
"a1b#c".toUpperCase(); // "A1B#C"
```

| 메서드                  | 역할         |
| -------------------- | ---------- |
| `toUpperCase()`      | 모두 대문자로    |
| `toLowerCase()`      | 모두 소문자로    |
| `equals()`           | 대소문자 구분 비교 |
| `equalsIgnoreCase()` | 대소문자 무시 비교 |
