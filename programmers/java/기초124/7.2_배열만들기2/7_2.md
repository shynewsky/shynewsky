# ⭐ List<Integer>

```java
// 패키지 import
import java.util.List;
import java.util.ArrayList;

// 리스트 선언
List<Integer> list;
// 리스트 구현
List<Integer> list = new ArrayList<>();
```

# ⭐ Queue<Integer>

```java
// 패키지 import
import java.util.Queue;
import java.util.ArrayDeque;

// 큐 선언
Queue<Integer> q;
// 큐 구현
Queue<Integer> q = new ArrayDeque<>();
```

---

## ⭐ isEmpty()

```java
// 리스트, 큐가 비어있는지 확인
q.isEmpty();
list.isEmpty();
```

## ⭐ size()

```java
// 리스트, 큐 요소 개수 반환
list.size();
q.size();
```

## ⭐ get(i)

```java
// 리스트의 i번쨰 요소 가져오기
list.get(i);
```

---

## ⭐ Collections.sort()

```java
// 리스트 오름차순 정렬
Collections.sort(list);
```
```java
// 리스트 내림차순 정렬
Collections.sort(list, Collections.reverseOrder());
```
