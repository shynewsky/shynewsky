# ⭐ startsWith()

- 문자열이 특정 문자열로 시작하는지(접두사인지)를 확인하는 String 클래스의 메서드

```java
boolean result = 문자열.startsWith(접두사);
```
```java
String s = "banana";

s.startsWith("ban");     // true
s.startsWith("b");       // true
s.startsWith("banana");  // true
s.startsWith("nan");     // false
s.startsWith("bananan"); // false
```
