# ⭐ toLowerCase()

- 문자열의 모든 알파벳을 소문자로 변환해서 새 문자열을 반환

```java
String s = "AbCdEf";
String lower = s.toLowerCase(); // "abcdef"
```

# ⭐ contains()

- 문자열 안에 연속된 부분 문자열이 포함되어 있는지 검사

```java
String s = "programmers";

s.contains("gram"); // true
s.contains("ramm"); // true
s.contains("gms");  // false (연속 아님)
```
