# ⭐ StringBuilder()

- 문자열을 효율적으로 “수정·추가”하기 위한 클래스
- Java의 String은 **불변(immutable)**이라서
한 번 만들면 내용을 바꿀 수 없음
- String을 +로 계속 붙일 때, 매번 새 객체 생성
- StringBuilder 클래스를 사용하면, 같은 객체를 계속 사용

```java
StringBuilder sb = new StringBuilder();

sb.append("a");
sb.append("b");
sb.append("c");
```

# ⭐ toString()

- StringBuilder → String으로 변환
- StringBuilder 자체는 String이 아님

```java
StringBuilder sb = new StringBuilder();
sb.append("hello");

// String result = sb;
String result = sb.toString();
```

---

## 자주 쓰는 메서드 정리

| 메서드 | 설명 |
|------|------|
| `append(x)` | 문자열 뒤에 추가 |
| `insert(i, x)` | i 위치에 삽입 |
| `delete(a, b)` | a ~ b-1 삭제 |
| `reverse()` | 문자열 뒤집기 |
| `toString()` | String으로 변환 |
| `length()` | 길이 반환 |
