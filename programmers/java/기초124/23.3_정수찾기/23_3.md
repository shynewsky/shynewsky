# ⭐ stream(num_list)

- IntStream 으로 변환
- 배열을 하나씩 흘려보내며 처리할 수 있는 스트림으로 만듦
- '배열을 반복문처럼 순회할 준비를 한다' 뜻

```java
Arrays.stream(num_list)
```

# ⭐ anyMatch(...)

- 스트림의 요소 중
조건을 만족하는 값이 하나라도 있으면 true
- 끝까지 없으면 false
- 조건을 만족하는 순간 즉시 종료 (break 효과)

```java
anyMatch(조건)
```

# ⭐ x -> x == n

- 람다식
- x : 배열의 각 원소
- x == n : 찾고 싶은 값인지 검사

```java
boolean test(int x) {
    return x == n;
}
```

# ⭐ 정리

- `num_list 안에 n과 같은 값이 하나라도 있으면 true`

```java
Arrays.stream(num_list).anyMatch(x -> x == n)
```
