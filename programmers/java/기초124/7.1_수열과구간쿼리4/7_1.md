# for (int[] q : queries)

- 자바의 향상된 for문(enhanced for / for-each)
- 배열이나 컬렉션의 모든 원소를 처음부터 끝까지 하나씩 꺼낼 때

```java
for (타입 변수 : 배열또는컬렉션) {
}
```

- queries : int[ ][ ] 타입 (2차원 배열)
- queries의 각 원소는 int[]
- q : 현재 반복에서 꺼낸 하나의 query 배열

```java
for (int[] q : queries)
```
```java
for (int i = 0; i < queries.length; i++) {
    int[] q = queries[i];
}
```
