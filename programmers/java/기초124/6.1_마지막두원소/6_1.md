# ⭐ System.arraycopy()

- 배열의 일부 또는 전체를 다른 배열로 복사하는 JVM 내장 메서드
- for문으로 하나씩 복사하는 것보다 보통 더 빠름
- 내부적으로 네이티브(최적화된) 코드 사용

```java
System.arraycopy(src, srcPos, dest, destPos, length);
```

| 매개변수      | 의미           |
| --------- | ------------ |
| `src`     | 원본 배열        |
| `srcPos`  | 원본 배열 시작 인덱스 |
| `dest`    | 대상 배열        |
| `destPos` | 대상 배열 시작 인덱스 |
| `length`  | 복사할 원소 개수    |

---

### 1️⃣ 전체 복사

```java
int[] a = {1, 2, 3};
int[] b = new int[4];

System.arraycopy(a, 0, b, 0, a.length); // b = [1, 2, 3, 0]
```

### 2️⃣ 부분 복사

```java
int[] a = {10, 20, 30, 40, 50};
int[] b = new int[3];

System.arraycopy(a, 1, b, 0, 3); // b = [20, 30, 40]
```

### 3️⃣ 배열 내에서 칸 이동

```java
int[] a = {1, 2, 3, 4, 5};

// 오른쪽으로 한 칸 밀기
System.arraycopy(a, 0, a, 1, 4); // a = [1, 1, 2, 3, 4]
```

### ⚠️ 주의사항

- 배열 크기 부족
- 타입 불일치

### 👍 사용 타이밍

- 배열에 원소 하나 추가하기
- 배열 앞에 원소 추가하기
- 배열의 일부만 잘라서 새 배열 만들기 (슬라이싱)
- 배열에서 특정 인덱스 제거하기
- 배열 이동(shift) / 밀기
