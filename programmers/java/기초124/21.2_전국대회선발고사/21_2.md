```java
List<int[]> candidates = new ArrayList<>();

```
```java
new int[]{rank[i], i}
```

# ⭐ sort((x, y) -> x[0] - y[0])

- 람다식(Lambda) 으로 만든 Comparator(비교 기준)

```java
x = {rank1, index1}
y = {rank2, index2}
// x, y는 리스트 안의 원소
```
```java
x[0] - y[0]
// x[0] → 첫 번째 값 = rank
// y[0] → 첫 번째 값 = rank
// rank 기준 오름차순 정렬
```
| 결과 | 의미   |
| -- | ---- |
| 음수 | x가 앞 |
| 양수 | y가 앞 |
| 0  | 동일   |
```java
[{5,3}, {2,7}, {4,1}] // 정렬 전
[{2,7}, {4,1}, {5,3}] // 정렬 후
```

```java
int[] first = candidates.get(0); // first = {2, 7}
```

```java
first = {2, 7} // 7
```

# ⭐ 정리

```java
candidates.add(new int[]{rank[i], i}); // 등수 + 학생번호 묶어서 저장
candidates.sort((x, y) -> x[0] - y[0]); // 등수(rank) 기준 오름차순 정렬
int a = candidates.get(0)[1]; // 1등 학생의 번호
```
