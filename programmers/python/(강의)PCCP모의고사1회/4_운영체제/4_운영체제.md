# 우선순위 큐

1. 선언
```python
import heapq
pq = []
```

2. 값 넣기
```python
heapq.heappush(pq, 3)
heapq.heappush(pq, 1)
heapq.heappush(pq, 2)
```

3. 값 꺼내기
```python
print(heapq.heappop(pq))  # 1
print(heapq.heappop(pq))  # 2
print(heapq.heappop(pq))  # 3
```

# 우선순위 여러조건

- 튜플 넣기
```python
heapq.heappush(pq, (우선순위1, 우선순위2, 값))
```

- 이 문제에 응용
```python
heapq.heappush(pq, (score, call_time, run_time)) # 넣을때
score, call, run = heapq.heappop(pq) # 꺼낼때
```
