# list

| 메서드                | 설명             | 예시                |
| ------------------ | -------------- | ----------------- |
| `append(x)`        | 맨 뒤에 추가        | `a.append(4)`     |
| `insert(i, x)`     | 특정 위치에 추가      | `a.insert(1, 10)` |
| `remove(x)`        | 값 제거           | `a.remove(2)`     |
| `pop()`            | 마지막 요소 제거 후 반환 | `a.pop()`         |
| `pop(i)`           | 특정 위치 제거       | `a.pop(0)`        |
| `index(x)`         | 위치 찾기          | `a.index(3)`      |
| `count(x)`         | 개수 세기          | `a.count(2)`      |
| `sort()`           | 정렬             | `a.sort()`        |
| `reverse()`        | 뒤집기            | `a.reverse()`     |
| `extend(iterable)` | 여러 개 추가        | `a.extend([4,5])` |
| `clear()`          | 전체 삭제          | `a.clear()`       |

# set

| 메서드              | 설명             | 예시                  |
| ---------------- | -------------- | ------------------- |
| `add(x)`         | 요소 추가          | `s.add(4)`          |
| `remove(x)`      | 요소 제거 (없으면 에러) | `s.remove(2)`       |
| `discard(x)`     | 요소 제거 (없어도 OK) | `s.discard(2)`      |
| `pop()`          | 임의 요소 제거       | `s.pop()`           |
| `clear()`        | 전체 삭제          | `s.clear()`         |
| `union()`        | 합집합            | `a.union(b)`        |
| `intersection()` | 교집합            | `a.intersection(b)` |
| `difference()`   | 차집합            | `a.difference(b)`   |

# dict

| 메서드                 | 설명          | 예시                  |
| ------------------- | ----------- | ------------------- |
| `get(key)`          | 값 가져오기      | `d.get("a")`        |
| `get(key, default)` | 없으면 기본값     | `d.get("b", 0)`     |
| `keys()`            | key 목록      | `d.keys()`          |
| `values()`          | value 목록    | `d.values()`        |
| `items()`           | key-value 쌍 | `d.items()`         |
| `update()`          | 여러 값 추가     | `d.update({"b":2})` |
| `pop(key)`          | key 삭제      | `d.pop("a")`        |
| `clear()`           | 전체 삭제       | `d.clear()`         |
