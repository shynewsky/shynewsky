# 100 나누기로 두자리씩 끊기

```cpp
    int answer = 0;
    
    for(int i=0; number != 0 ; i++){ 
        answer += number % 100; 
        number /= 100;

        // 100으로 나눈 나머지는 십의자리와 일의자리이고, 몫은 천의자리와 백의자리이다
        // 나머지는 answer에 계속 누적하고, number는 몫으로 대체하여 또 다시 100으로 나눈다
        // 몫이 0이 되어 더이상 나누지 않으면 루프를
```
